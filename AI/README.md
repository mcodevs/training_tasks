# AI Configuration Directory

This directory contains configuration files and rules for AI-assisted development tools.

## Contents

- **\_cursorrules**: specific instructions and architectural rules for the project. This ensures that AI assistants generate code that adheres to the project's strict standards, including:
  - BLoC architecture
  - Elixir navigation
  - Dependency injection patterns
  - Folder structure
  - Coding style and conventions

## Usage

These files are automatically read by supported AI tools to provide context-aware code generation and suggestions that match the existing codebase.
